Welcome to chemlab website!

This is my final project for CS50's Web Programming with Python and JavaScript course.

In this project, I've built a web application for handling sample records and analysis results of a chemistry laboratory that performs gasoline and gas oil analysis. Laboratory staff can add/edit samples, enter analysis results of samples, and add/edit customers of the laboratory. Laboratory manager can also add/edit analysis methods and products to be analysed. Customers of the laboratory can see analysis results of their samples.

On the samples page laboratory staff can add new samples, edit existing samples or delete existing samples. Laboratory staff can also enter the analysis results of the samples on the analysis results page. Customers of the laboratory can be added, edited and deleted on the customers page.

Customers can see analysis results of their results on the analysis results page when they are logged in.

On the site administration page laboratory manager can edit analysis methods and products to be analysed using Django's built-in admin application.

Customers can only see links to homepage and analysis results page on the navigation bar. In addition to these, links to samples and customers pages are visible to laboratory staff. Laboratory manager can see all of the links on the navigation bar.

Sample, result, customer, product and analysis method data are stored in a PostgreSQL database hosted by Heroku.

This web application is mobile-responsive.
